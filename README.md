# struktury
Projekt zawiera szablony niektórych struktur.<br>
Projekt będzie rozwijany poprzez dodawanie kolejnych struktur.<br>
Aktualnie projekt pozwala korzystać z:
- drzewa BST
- drzewa AVL
- drzewa Splay

<br>Używanie struktur nie wymaga kompilacji<br>
Należy dostarczyć funkcję compare dla własnej struktury (dostarczone są dla int i dla std::string),<br>
aby to zrobić należy w pliku compare.hpp w folderze include należy dopisać specjalizację szablonu dla własnej struktury.<br>
Po zapewnieniu funkci trzeba jedynie dołączyć do swojego projektu plik struktury.hpp oraz ścieżkę do tego pliku.
